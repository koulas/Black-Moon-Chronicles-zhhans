NEW GAME
Select this option to start a new game. There are three types of games to choose from :

Campaigns
Select this option to start a campaign. There are five of them: " Initiation ", " Empire Campaign ", " Justice Campaign ", " Light Campaign " and " Black Moon Campaign ", but only " Initiation " is available on installing the game. In this campaign, you are not involved in fighting any enemy regiment or in any resource management : you only have an overall outlook on the possibilities of the game, and you are mostly supposed to become familiar with the interface.
Once this first campaign is over, however, the other four will become available to you. You will then have to manage your resources and face enemy regiments.

 

Skirmish
Select this option to play one card which you have created in the level editor. In this mode, you are facing the enemy's army in a single battle, which can but end with the extermination of one of the two camps.

 
Special missions
This option offers you a few additional single-player missions.

 

LOADING A GAME
Select this option to load a game you have saveguarded previously. The names of the games listed appear on the screen : click on the game you wish to play and then " Load " it to retrieve it.

 

OPTIONS

Resolution
This option alllows you to change the graphics quality of the game. The higher the resolution, the slower the game. The lower the resolution, the faster the game.

Sound
This option allows you to change the volume of the music and sound effects of the game.
Parental surpervision
This option allows you to remove certain graphics effects of the game (blood, etc.).

 

BESTIARY
Select this option to gain access to the presentation of the different characters on each side.

 

CREDITS
Select this option to consult the game's credits.

 

QUIT
Select this option to quit the game " Black Moon Chronicles ".
